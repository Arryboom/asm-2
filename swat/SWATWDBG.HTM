Windows Debugging

386SWAT runs as a debugger under Windows 3.x and 95/98 when accompanied by its VxD. This file (SWATVXD.EXE) should be in the same directory as the debugger file (386SWAT.LOD) and is loaded automatically when Windows starts. To take advantage of the information provided by the VxD, you should have a monochrome monitor attached to your system. For more details about the VxD including how to configure it, see the file SWATVXD.DOC.
